BTLE Control

Mode: OFF

Device Value: 0

Display Value: ---

Usage:

  • Device only works 'in mount'
  • Battery will be consumed when active and can be charged at any time 
  • Long press to switch on
  • Scroll wheel with mouse to adjust ND value
  • Click to toggle ND adjust mode
  • Long press while on to toggle BLE mode. Input from 'remote' accepted only
  • Double click to toggle interface lock.
  • Hold button >5s for power down.
